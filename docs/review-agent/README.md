# Review Agent Data

Machine-readable governance records for WealthHouse documentation-agent.

- `bug-registry.json` is the canonical bug registry.
- `review-history.json` stores one review object per merged PR.
- `change-log.json` stores normalized release entries.
- `module-impact-index.json` stores aggregate module impact counters.
- `config.json` stores module/risk pattern configuration.
- `audit-logs/` stores run-by-run trace logs.
